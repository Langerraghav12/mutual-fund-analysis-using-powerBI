> using Python data analysis and an interactive Power BI dashboard.

## 🚀 How to Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Fetch the dataset (one-time, ~10–15 minutes)
```bash
python generate_dataset.py
```
This calls the [mfapi.in](https://www.mfapi.in) API for 2500+ schemes and saves to `data/mutual_funds_raw.csv`.

### 3. Run the full analysis
```bash
python mutual_fund_analysis.py
```

Or open the notebook:
```bash
jupyter notebook mutual_fund_analysis.ipynb
```

## 🧠 Python Analysis Pipeline

### 1. Data Cleaning
- Removed rows with missing NAV, 1Y return, or 3Y return
- Filtered out invalid/negative NAV values
- Removed extreme return outliers (>500% or <-100%)
- Standardized text fields (fund house, category)

### 2. Descriptive Statistics
- `mean`, `median`, `std`, `min`, `max` for all numeric fields
- Fund count distribution by scheme type and category

### 3. Normalization
- Applied `MinMaxScaler` from `sklearn.preprocessing`
- All features scaled to [0, 1] before scoring

### 4. Custom Scoring Formula
```
Score = 0.40 × return_3y_norm
      + 0.30 × return_1y_norm
      + 0.15 × fund_age_norm
      + 0.15 × latest_nav_norm
```
Only funds with **1-Year Return > 0** are eligible.

### 5. Top 30 Output
Funds ranked by composite score → Top 30 exported to Excel (2 sheets).
